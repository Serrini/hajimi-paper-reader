# Contributing

Thanks for your interest in Hajimi Studio.

## Before opening a pull request

- open an issue for bugs, regressions, or major feature proposals
- keep pull requests focused and small when possible
- include clear reproduction steps for bug fixes
- update documentation when behavior changes

## Local setup

```bash
npm install
npm run dev
```

For packaging and release notes, see [docs/manual-release.md](docs/manual-release.md).
