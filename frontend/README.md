# Hajimi Paper Reader Frontend

Hajimi Paper Reader 的前端界面，基于 React + TypeScript + Ant Design 构建。

## 功能特性

- 📁 文件管理：上传、查看、删除文件
- 🗂️ 知识库管理：创建、管理知识库
- 💬 智能问答：基于RAG的智能问答系统
- 🤖 Agent对话：基于LangChain的AI Agent
- 📊 数据统计：系统使用统计

## 技术栈

- React 18
- TypeScript
- Ant Design 5
- React Router 6
- Axios

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

## 访问地址

- 开发环境：http://localhost:3000
- 生产环境：根据部署配置
